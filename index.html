01 — About

Analyst, builder, problem-solver.

I'm an AI Data Analyst — I bring retrieval-augmented generation, LLMs, and machine learning into the analytics workflow so teams can ask questions of their data instead of hunting through dashboards and documents.

That AI layer sits on top of the fundamentals: Python for automation, SQL for modeling, Power BI and Tableau for dashboards, and Azure / Databricks for cloud pipelines. The goal is the same as any good analytics work — faster, clearer decisions.

  • ◆ Ground AI in real data — no hallucinated answers, cite the source.
  • ◆ Automate what doesn't need a human — ship dashboards for what does.
  • ◆ Write SQL and Python you can read six months later.
